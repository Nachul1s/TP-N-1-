¿A que llamamos elemento HTML? ¿Cual es su anatomia? Explicar sus partes

¿Que es un atributo? Dar 2 ejemplos

1- HTML Consiste en el lenguaje marcado de hipertexto, es el codigo que se usa para estructurar y desplegar la pagina web

2- La etiqueta marca el inicio y final de un elemento, da igual de que manera se escriba

3- El elemento es la etiqueta de apertura, seguida del contenido, seguida de la etiqueta de cierre. Sus principales partes de elementos son:

La etiqueta de apertura que consiste en el nombre del elemento encerrado en signo mas o menos.

La etiqueta de cierre que es igual a la etiqueta de apertura excepto que incluye una barra antes del nombre de la etiqueta

Sus contenido son lo que llevan dentro.

4- Un atributo agrega. complementa al elemento asi para que tenga un comportamiento diferente al esperado. Un ejemplo de atributo es "alt", que especifica una descripcion de texto en una imagen. por ejemplo att= "el icono de brave"

Mi segundo ejemplo es el atributo "width" que especifica el ancho de pixeles de la pagina. Por ejemplo width="500".
